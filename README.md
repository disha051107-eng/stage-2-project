# Movie Recommendation System

## About the Project

This project is a **Movie Recommendation System** built using **Python, NumPy, and Pandas**.

It recommends movies to users based on the ratings given by other users with similar interests.

In this project, I created the recommendation system **without using Scikit-learn**. All calculations are done manually using Python and NumPy.

---

## Features

- Load movie ratings data
- Create a user-movie matrix
- Find similar users
- Predict movie ratings
- Recommend the best movies
- Built without machine learning libraries

## How to Run

1. Download the project.
2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Run the program:

```bash
python recommendation.py
```

4. Enter a username to get movie recommendations.
